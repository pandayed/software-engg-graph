Python has several built-in data types for storing and manipulating data.

## Numeric Types

- **int**: Integer numbers (`42`, `-10`).
- **float**: Floating-point numbers (`3.14`, `-0.5`).
- **complex**: Complex numbers (`3+4j`).

## Sequence Types

- **str**: Immutable sequence of characters (`"hello"`).
- **list**: Mutable ordered sequence (`[1, 2, 3]`).
- **tuple**: Immutable ordered sequence (`(1, 2, 3)`).

## Mapping Type

- **dict**: Key-value pairs (`{"name": "Alice", "age": 30}`).

## Set Types

- **set**: Unordered collection of unique elements (`{1, 2, 3}`).
- **frozenset**: Immutable set.

## Boolean Type

- **bool**: `True` or `False`.

## None Type

- **None**: Represents absence of value.
